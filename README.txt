
                                   PyAAF

What is it?
-----------

Python Bindings for the Advanced Authoring Format (AAF) written in Boost.Python


Requirements
------------

AAF SDK - http://sourceforge.net/projects/aaf/files/AAF-src
boost - http://www.boost.org/
    with Boost.Python http://www.boost.org/doc/libs/1_53_0/libs/python/doc/index.html
    
Optional (for code generation)

gccxml - https://github.com/gccxml/gccxml
pyste (include in with boost libs/python/pyste)
PyYAML - http://pyyaml.org/
